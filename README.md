# Support System

**Help your favorite creators, effortlessly.**

Support System is a link collection app that organizes your shopping by merchant and helps you remember to use creator referral/affiliate codes at checkout.

## The Problem

You follow creators who have referral codes for stores you shop at, but:
- You forget to use their codes when checking out
- You don't remember which creators have codes for which stores
- There's no easy way to track your shopping patterns

## The Solution

Support System creates a simple workflow:
1. **Save** - Share shopping links to the app
2. **Organize** - Links auto-group by merchant/store
3. **Assign** - Set your preferred creator (benefactor) for each store
4. **Support** - Get reminded of the code when you shop

## Features

### MVP
- Save links from any app via share sheet or paste
- Auto-extract merchant/domain from URLs
- View links grouped by merchant
- Assign benefactor codes to merchants
- See benefactor reminder when opening links

### Future
- **V1**: Browser extension to auto-apply codes at checkout
- **V2**: Community/scraped codes as fallbacks
- **V3**: Creator dashboard for verified profiles
